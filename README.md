Application link: https://csmach1-envlog-scuba-20260701.streamlit.app/

# CS-MACH1
CS-MACH1 (Marine Citizen Science Data Horizon) is a Horizon Europe project that aims to strengthen marine citizen science by improving the accessibility, quality, trustworthiness, and interoperability of citizen-generated marine data. The project is establishing a Marine Citizen Science Data Network (MCSDN) to connect citizen science initiatives, researchers, technology developers, and data experts. It promotes FAIR data practices, develops standards and best practices for data collection and management, supports the use of cost-effective sensing technologies, and facilitates the integration of citizen science observations into European marine data infrastructures such as EMODnet. The project also demonstrates the scientific and societal value of marine citizen science through real-world applications, Digital Twin Ocean validation, and stakeholder engagement.
